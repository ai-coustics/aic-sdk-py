    initialize(sample_rate, channels, frames) ¶

    Allocate internal DSP state.

    Parameters:

    • sample_rate (int) –

      Input/output sample rate in Hz.

    • channels (int) –

      Number of channels in the processed audio (e.g., 1 for mono, 2 for stereo).

    • frames (int) –

      Block length in frames for streaming. Use :py:meth:optimal_num_frames for a recommended value.

    Raises:

    • RuntimeError –

      If the underlying SDK rejects the configuration.

    process(pcm, *, channels=None) ¶

    Enhance pcm in-place using planar processing (convenience pass-through).

    Parameters:

    • pcm (ndarray) –

      Planar 2-D array of shape (channels, frames) Data must be float32 in the linear -1…+1 range. Any non-conforming array is copied to a compliant scratch buffer.

    • channels (int | None, default: None ) –

      Override channel count auto-detected from pcm. Rarely needed.

    Returns:

    • ndarray –

      The same array instance (modified in-place) or a contiguous copy if a dtype/stride conversion had been necessary.

    process_interleaved(pcm, channels) ¶

    Enhance pcm in-place using interleaved processing (convenience pass-through).

    Parameters:

    • pcm (ndarray) –

      Interleaved 1-D array of shape (frames,) containing interleaved audio data Data must be float32 in the linear -1…+1 range. Any non-conforming array is copied to a compliant scratch buffer.

    • channels (int) –

      Number of channels in the interleaved data.

    Returns:

    • ndarray –

      The same array instance (modified in-place) or a contiguous copy if a dtype/stride conversion had been necessary.

    Returns:

    • int –

    set_parameter(param, value) ¶

    Update an algorithm parameter.

    Parameters:

    • param (AICParameter) –

      Parameter enum value. See :py:class:aic._bindings.AICParameter.

    • value (float) –

      New value for the parameter (float).

    Raises:

    • RuntimeError –

      If the parameter is out of range or the SDK call fails.

    Available neural model variants.

    LEGACY_L = 4 class-attribute instance-attribute ¶

    Legacy large model - 512-frame, ~10.67ms latency.

    LEGACY_S = 5 class-attribute instance-attribute ¶

    Legacy small model - 256-frame, ~5.33ms latency.

    QUAIL_L = 0 class-attribute instance-attribute ¶

    Large model - highest quality, higher compute usage.

    QUAIL_S = 1 class-attribute instance-attribute ¶

    Small model - balanced quality and speed.

    QUAIL_XS = 2 class-attribute instance-attribute ¶

    Extra small model - fastest, lower quality.

    QUAIL_XXS = 3 class-attribute instance-attribute ¶

    Ultra small model - lowest latency, minimal compute.

    Algorithm parameters adjustable at runtime.

    BYPASS = 0 class-attribute instance-attribute ¶

    Bypass processing (0.0 off → 1.0 on).

    ENHANCEMENT_LEVEL = 1 class-attribute instance-attribute ¶

    Overall enhancement strength (0.0 … 1.0).

    ENHANCEMENT_LEVEL_SKEW_FACTOR = 2 class-attribute instance-attribute ¶

    Skew factor for non-linear enhancement mapping.

    NOISE_GATE_ENABLE = 4 class-attribute instance-attribute ¶

    Enable/disable noise gate (0.0 off → 1.0 on).

    VOICE_GAIN = 3 class-attribute instance-attribute ¶

    Additional gain applied to detected speech (linear).
